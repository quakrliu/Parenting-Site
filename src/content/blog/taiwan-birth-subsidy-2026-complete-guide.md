# 台灣 2026 生育補助完整攻略：每個寶寶 NT$100,000 怎麼領？

恭喜新手爸媽！從 2026 年 1 月 1 日起，台灣中央政府大幅擴充生育支持政策，**每位新生兒可領 NT$100,000 統一生育給付**，雙胞胎則為 NT$200,000，且**不受勞保、健保資格限制**，幾乎所有在台出生的新生兒都適用。

加上各縣市的加碼補助，部分地區的家庭最高可領到超過 **NT$150,000** 甚至更多。這篇攻略幫你整理清楚每一筆錢怎麼領、要準備什麼文件、什麼時候申請。

---

## 一、中央政府統一生育給付 NT$100,000

### 什麼是「統一生育給付」？

這是 2026 年新制的核心。過去台灣生育補助分散在不同制度（勞保生育給付、全民健保、地方補助），資格各異，讓許多家庭漏領。新制整合為一個「統一給付」窗口：

| 項目 | 金額 |
|------|------|
| 單胞胎新生兒 | **NT$100,000** |
| 雙胞胎 | **NT$200,000** |
| 三胞胎（以上） | **NT$300,000**（每位 NT$100,000） |

### 申請資格

- 新生兒於 **2026年1月1日（含）以後**出生
- 新生兒出生時具有**中華民國國籍**（或父母一方為本國籍）
- 在台灣辦理**出生登記**

> **不需要：** 勞保資格、健保年資、收入限制

### 如何申請？

**申請途徑一：戶政事務所「一站式服務」（推薦）**

攜帶以下文件至戶政事務所辦理出生登記時，**同步申請生育給付**，最方便：

- 新生兒出生證明書（醫院開立）
- 父母雙方身分證
- 父母戶口名簿（或戶籍謄本）
- 父母其中一方的銀行存摺封面（匯款用）

**申請途徑二：線上申請**

透過「全國性生育給付整合平台」（請至衛福部官網查詢最新連結）線上申辦，需準備電子文件。

### 何時可以領到？

- 審核通過後，通常 **30個工作天內**匯入指定帳戶
- 如資料齊全，部分縣市可於 **2週內**核撥

### 申請期限

建議於新生兒出生後 **6個月內**申請，超過2年可能失效（以主管機關公告為準）。

## 二、勞保生育給付（另計，可與統一給付並領）

除了中央統一給付，**勞保被保險人**還可額外申請勞保生育給付：

| 項目 | 計算方式 |
|------|---------|
| 生育給付 | 投保薪資 × 2 個月 |
| 雙胞胎 | 投保薪資 × 4 個月 |

**申請資格：**
- 媽媽（或爸爸，若媽媽非勞保被保險人）在職並持續投保勞保
- 分娩或流產當日仍有勞保資格

**申請方式：**
- 向勞保局申請，或透過雇主轉申報
- 文件：出生證明、勞保被保險人資格證明

> 💡 **統一給付 + 勞保給付可以同時領**，別漏掉！

以下為各縣市政府「加碼」的生育補助，**疊加在中央NT$100,000之上**。各縣市政策隨年度調整，請以縣市政府官方公告為準：

|------|------|------|------------|------|
| 台北市 | +NT$2萬 | +NT$3萬 | +NT$5萬 | 另有育兒津貼 |
| 新北市 | +NT$1萬 | +NT$2萬 | +NT$3萬 | |
| 桃園市 | +NT$1萬 | +NT$1.5萬 | +NT$2萬 | |
| 台中市 | +NT$2萬 | +NT$3萬 | +NT$4萬 | |
| 台南市 | +NT$1萬 | +NT$2萬 | +NT$3萬 | |
| 高雄市 | +NT$1萬 | +NT$2萬 | +NT$3萬 | |
| 宜蘭縣 | +NT$3萬 | +NT$4萬 | +NT$5萬 | 偏鄉加碼 |
| 花蓮縣 | +NT$3萬 | +NT$5萬 | +NT$7萬 | 偏鄉加碼 |

> **注意：** 以上數字為概估，各縣市政策可能隨時調整。**請務必至您所在縣市政府官方網站查詢最新公告**，或撥打各縣市市政府服務電話確認。

### 如何申請縣市加碼補助？

大多數縣市加碼補助需**另外申請**，不會自動核撥。通常可：

1. 至戶籍所在地的**區公所、鄉鎮市公所**申請
2. 部分縣市提供**線上申辦**（如台北市e-services平台）

所需文件通常與中央補助相同，但可能需要額外的**戶籍謄本**以證明居住年限。

## 四、育兒津貼：每月持續補助

除了一次性生育給付，政府也提供**持續性**的育兒支持：

### 育兒津貼（未就學幼兒）

- **0-2歲**未送托（自己照顧）：每月 **NT$5,000**
- **2-5歲**未就讀幼兒園：每月 **NT$3,500**（各縣市可能加碼）

### 就讀公立幼兒園或準公共幼兒園

- 就讀公立幼兒園：每月收費**約NT$2,500-3,500**（補助後）
- 就讀準公共化合作園：每月家長負擔**不超過NT$3,000**

### 第三胎以上：加碼育兒津貼

- 第三胎（以上）：育兒津貼再加碼 NT$1,000/月

## 五、產假、陪產假、育嬰留職停薪薪資補貼

### 媽媽：產假相關

| 項目 | 天數 | 薪資 |
|------|------|------|
| 一般產假 | **56天** | 全薪（受僱者） |
| 生產後住院期間 | 另計 | 由健保給付 |

### 爸爸：陪產假/陪產檢假

- **陪產及陪產檢假：7天**（全薪，由雇主支付）

### 育嬰留職停薪

- 孩子3歲前，爸媽各可申請最長 **6個月**（合計最長1年）的留職停薪
- 留停期間可領「育嬰留職停薪津貼」：投保薪資的 **80%**（最長6個月）

## 六、完整申請清單：一步一步來

### Step 1｜出生後7天內

- [ ] 醫院取得**出生證明書**（至少申請3份）
- [ ] 為寶寶辦理**出生登記**（戶政事務所，可委託辦理）

### Step 2｜出生後1個月內

- [ ] 同步申請**中央統一生育給付 NT$100,000**（戶政事務所一站式）
- [ ] 若符合資格，申請**勞保生育給付**
- [ ] 申請各縣市**加碼補助**（區公所或線上）

### Step 3｜出生後3個月內

- [ ] 申請**育兒津貼**（若選擇自己照顧）
- [ ] 確認是否符合**育嬰留職停薪**資格，與雇主協商

### 別忘了的小事

- [ ] 為寶寶辦理**全民健保**加保（出生後60天內）
- [ ] 申請**兒童預防保健**服務（7次免費健檢）
- [ ] 查詢縣市是否有**新生兒訪視**服務（免費衛教師到府指導）

## 常見問題 Q&A

**Q：我是全職媽媽（家管），也可以領NT$100,000嗎？**
A：可以。2026年新制的統一生育給付**不需要勞保或就業資格**，全職媽媽同樣適用。

**Q：自然流產也有補助嗎？**
A：統一生育給付限活產新生兒。但勞保的**生育給付**，妊娠滿20週（含）流產者，可領1個月投保薪資，未滿20週者則視情況，請向勞保局確認。

**Q：外籍配偶的孩子可以領嗎？**
A：若孩子具有中華民國國籍，即可申請。具體資格請洽戶政事務所確認。

**Q：雙薪家庭，爸爸媽媽都有勞保，生育給付可以兩個人都領嗎？**
A：統一給付以新生兒為申請單位，一個孩子領一次（NT$100,000）。勞保生育給付：若媽媽符合資格，由媽媽申請；若媽媽非勞保被保險人，才由爸爸申請，兩人不能重複。

**Q：如果我的孩子在2025年底出生，可以溯及適用新制嗎？**
A：不行。統一生育給付適用對象為2026年1月1日以後出生的新生兒，2025年底出生者依舊適用舊制。

## 所有補助加起來能領多少？

以台中市雙薪家庭（父母皆有勞保，投保薪資各NT$40,000）生育第一胎為例：

| 補助項目 | 金額 |
|---------|------|
| 中央統一生育給付 | NT$100,000 |
| 台中市加碼補助 | NT$20,000 |
| 媽媽勞保生育給付（40,000×2） | NT$80,000 |
| **合計** | **NT$200,000** |

再加上往後每月NT$5,000的育兒津貼（0-2歲），2年合計可再累積 NT$120,000。
